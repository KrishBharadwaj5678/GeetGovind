# 🕉️ Geet Govind

A devotional Flutter application dedicated to **Shri Jagannath Ji**, featuring sacred texts, divine images, and soulful bhajans.
The app provides a peaceful and immersive spiritual experience for devotees.

![GeetGovindDemo](https://github.com/KrishBharadwaj5678/GeetGovind/raw/main/GeetGovindDemo.jpeg)

## ✨ Features

| Feature             | Description                                                                           |
| ------------------- | ------------------------------------------------------------------------------------- |
| 🏠 Home Screen      | Displays **Geet Govind** for reading and devotion                                     |
| 🖼️ Gallery         | Beautiful collection of **Shri Jagannath Ji images**                                   |
| 🎵 Bhajans          | Collection of devotional **bhajans of Jagannath Ji**                                  |
| ▶️ Music Player     | Built-in player to listen to bhajans                                                  |
| 🎨 Clean UI         | Simple, elegant, and user-friendly design                                             |
| 📱 Mobile Optimized | Smooth performance on Android devices                                                 |

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/KrishBharadwaj5678/GeetGovind.git
```

### 2️⃣ Navigate to project directory

```bash
cd GeetGovind
```

### 3️⃣ Install dependencies

```bash
flutter pub get
```

### 4️⃣ Run the app

```bash
flutter run
```
